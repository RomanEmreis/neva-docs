# Глоссарий терминов

Данный глоссарий содержит ключевые термины, используемые в документации Neva, и их переводы на русский язык. Термины применяются единообразно во всей документации.

| Английский термин | Русский перевод | Примечание |
|-------------------|-----------------|------------|
| Model Context Protocol (MCP) | Протокол модельного контекста (MCP) | Аббревиатура MCP не переводится |
| server | сервер | |
| client | клиент | |
| tool | инструмент | В контексте MCP — функция, вызываемая языковой моделью |
| resource | ресурс | Данные, предоставляемые сервером клиенту |
| prompt | запрос | Шаблон сообщения для языковой модели |
| handler | обработчик | Функция, обрабатывающая запрос |
| middleware | промежуточный обработчик | Перехватчик в цепочке обработки запросов |
| sampling | сэмплирование | Запрос к LLM на генерацию текста |
| elicitation | получение данных | Запрос структурированных данных у пользователя |
| transport | транспорт | Протокол передачи данных (stdio, HTTP) |
| stdio | stdio | Стандартный ввод/вывод; не переводится |
| dependency injection (DI) | внедрение зависимостей (DI) | |
| singleton | синглтон | Жизненный цикл сервиса: один экземпляр на всё время работы |
| scoped | с областью видимости / scoped | Жизненный цикл сервиса: один экземпляр на запрос |
| transient | временный / transient | Жизненный цикл сервиса: новый экземпляр при каждом разрешении |
| feature flag | флаг компонента | Флаг Cargo для включения/отключения функциональности |
| attribute macro | атрибутный макрос | Макрос вида `#[tool]`, `#[resource]`, `#[prompt]` |
| token | токен | JWT-токен или токен прогресса/контекста |
| schema | схема | JSON Schema для входных/выходных данных |
| task | задача | Долгосрочная асинхронная операция |
| logging | логирование | |
| error handling | обработка ошибок | |
| progress notification | уведомление о прогрессе | |
| root | корневой каталог | Путь в файловой системе, предоставляемый клиентом серверу |
| session | сессия | |
| payload | полезная нагрузка | Данные запроса или ответа |
| completion | автодополнение | Предложение вариантов ввода для аргументов |
| annotation | аннотация | Произвольные метаданные, прикреплённые к ресурсу или инструменту |
| runtime | среда выполнения | |
| bearer token | токен Bearer | JWT-токен для аутентификации HTTP |
| structured output | структурированный вывод | Ответ инструмента в формате JSON |
| TTL (time-to-live) | TTL (время жизни) | Таймаут задачи |
